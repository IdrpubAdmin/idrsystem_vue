농심 메가마트 쇼핑몰 구축

  • 메가마트는 전국에 15개 점포를 보유한 대형마트이며,
    메가마트 온라인 쇼핑몰은 지역 마트 기반의 온라인 할인마트 시스템임
  • 메가마트의 상품관리시스템에서 각 매장별 가격과 재고정보,
    이벤트 정보를 실시간 인터페이스 받음
  • 주문 정보는 매장 주문관리시스템으로 인터페이스 하여 매장에서
    Pickig -> Packing -> Shipping 처리
  • 주문 입력 시 배송 가능 시간을 고객이 직접 선택
  • SAP Hybris Commerce Platform으로 3개월만에 1단계 오픈
  • 2단계는 매장기반이 아닌 협력사 배송시스템 확대 구축
